# Contributing

Thanks for contributing to `@huh-david/bmp-js`.

## Requirements

- Node.js 22+
- pnpm 10+

## Setup

```bash
pnpm install
```

## Local quality gate

```bash
pnpm check
```

This runs formatting checks, linting, type checks, build, and tests.

## Changesets

We use Changesets for release management.

```bash
pnpm changeset
```

Add a changeset for user-visible changes.

## Pull requests

- Keep PRs focused and small when possible.
- Include tests for behavior changes.
- Update docs if API behavior changes.
