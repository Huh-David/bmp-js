# Contributing

Contributions are welcome.

Main resources:

- [Repository](https://github.com/Huh-David/bmp-js)
- [Contributing Guide](https://github.com/Huh-David/bmp-js/blob/master/CONTRIBUTING.md)
- [Issue Tracker](https://github.com/Huh-David/bmp-js/issues)

Local setup:

```bash
pnpm install
pnpm check
```

For release-relevant changes, add a changeset:

```bash
pnpm changeset
```
